## 实现步骤

1. **修改electron-builder.json5文件**

   * 在NSIS配置部分添加`createDesktopShortcut`选项，设置为`"prompt"`以提示用户添加桌面快捷方式

   * 确保`createStartMenuShortcut`选项设置为`true`以添加系统启动菜单快捷方式

2. **重新构建应用程序**

   * 运行`npm run build`命令重新生成安装程序

## 具体修改

在`electron-builder.json5`文件的NSIS配置部分添加以下选项：

```json5
"nsis": {
  "oneClick": false,
  "perMachine": false,
  "allowToChangeInstallationDirectory": true,
  "deleteAppDataOnUninstall": false,
  "createDesktopShortcut": "prompt",
  "createStartMenuShortcut": true
}
```

这样修改后，安装程序会在安装过程
