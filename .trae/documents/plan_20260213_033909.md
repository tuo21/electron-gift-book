## 全屏适配实现计划

### 核心思路
使用 CSS `transform: scale()` 实现等比放大，保持原有布局不变，只对主内容区进行缩放。

### 实现步骤

#### 1. 创建缩放组合式函数 `useFullscreenScale.ts`
- 定义基准设计尺寸（宽度 1440px）
- 监听窗口 resize 事件
- 计算缩放比例：`scale = Math.max(1, window.innerWidth / baseWidth)`
- 提供缩放比例响应式变量

#### 2. 修改 `App.vue`
- 引入 `useFullscreenScale` 组合式函数
- 对 `.main-content` 区域应用缩放样式
- 使用 CSS 变量动态设置 transform-origin 和 scale

#### 3. 更新 `theme.css`
- 添加缩放相关的 CSS 变量

### 技术要点
- **只放大不缩小**：`scale = Math.max(1, ...)` 确保最小比例为 1
- **transform-origin: top center**：从顶部中心开始缩放
- **主内容区缩放**：只对 `.main-content` 应用缩放，header 保持原样
- **响应式更新**：窗口大小变化时自动重新计算缩放比例
