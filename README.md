## 主要功能

### 📝 礼金记录管理
- **添加记录**：支持录入姓名、金额、支付方式、物品描述和备注
- **编辑修改**：随时修改已添加的记录
- **删除管理**：支持软删除，保留删除历史
- **多种支付方式**：现金、微信、内收等多种支付类型

### 📊 数据统计分析
- **实时统计**：总人数、总金额自动计算
- **分类统计**：按支付方式分类统计金额
- **统计详情**：查看详细的统计数据

### 📤 导入导出功能
- **Excel导出**：导出为Excel表格，适合数据分析
- **PDF导出**：导出为传统礼金簿样式的PDF，适合打印存档
- **批量导入**：支持从外部数据源导入记录

### 🔍 智能搜索
- **模糊搜索**：支持搜索姓名、备注、物品描述
- **快速定位**：点击搜索结果直接跳转到对应记录

### 📚 多礼金簿管理
- **新建礼金簿**：为不同场合创建独立的礼金簿
- **打开已有**：快速打开最近使用的礼金簿
- **文件管理**：管理礼金簿文件，支持删除不需要的文件

### 🎨 主题定制
- **喜庆主题**：传统红色喜庆风格
- **洒金宣纸**：模拟传统宣纸肌理效果
- **竖排展示**：PDF导出时支持传统竖排格式

### ⏰ 农历日期显示
- **实时更新**：显示当前农历日期
- **传统节日**：支持重要农历节日显示

## 使用说明

1. **启动应用**：双击应用图标启动电子礼金簿
2. **新建礼金簿**：在启动页输入事件名称，选择主题，点击「开始」
3. **录入记录**：在右侧表单填写姓名、金额等信息，点击「保存」
4. **查看记录**：在左侧礼金簿区域查看所有记录
5. **导出数据**：点击顶部「导出」按钮，选择Excel或PDF格式
6. **搜索记录**：点击顶部「搜索」按钮，输入关键词进行搜索
7. **查看统计**：点击顶部「统计」按钮，查看详细统计数据
