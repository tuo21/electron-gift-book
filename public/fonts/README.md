# 字体文件说明

本目录用于存放电子礼金簿项目所需的字体文件。

## 字体规范

根据项目字体规范，各类文本使用以下字体：

| 文本类型 | 字体 | 说明 |
|---------|------|------|
| 姓名文本 | 演示春风楷 | 礼金簿中宾客姓名 |
| 大写金额 | 演示春风楷 | 礼金大写金额（壹贰叁肆...） |
| 固定文字 | 宋体 (SimSun) | 标签、按钮等固定文字内容 |
| 变量文字 | 楷体 (KaiTi) | 备注、描述等可变文字内容 |

## 需要添加的字体文件

请将以下字体文件放入本目录：

1. **XuandongKaishu.ttf** - 用于姓名和大写金额显示
   - 这是本项目的主要特色字体
   - 如未安装，系统会自动回退到楷体

## 字体回退机制

如果"演示春风楷"字体不可用，系统会按以下顺序回退：
1. 演示春风楷
2. 楷体 (KaiTi)
3. 华文楷体 (STKaiti)
4. 系统默认衬线字体

## 跨平台兼容性

- **Windows**: 支持 SimSun、KaiTi、STKaiti
- **macOS**: 支持 STSong、STKaiti
- **Linux**: 建议安装 Noto Serif SC 字体包

## PDF 导出

PDF 导出时会自动嵌入字体，确保打印效果一致。
